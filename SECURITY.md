# Security Policy

## Supported versions

Security fixes are provided for the latest published version of
`@ngelik/brain-hands` and the current `main` branch.

## Reporting a vulnerability

Do not open a public issue or pull request for a suspected vulnerability.
Use GitHub's private vulnerability reporting for this repository:

https://github.com/ngelik/brain-hands/security/advisories/new

Include the affected version or commit, impact, reproduction steps, and any
suggested mitigation. Remove credentials, personal data, private repository
content, and unrelated logs from the report.

The maintainer will acknowledge a complete report as soon as practical,
validate it privately, and coordinate remediation and disclosure. Please do not
publish details before a fix or agreed disclosure date is available.
