# Support

For usage questions, search the README and existing issues first. If the answer
is not available, open a GitHub issue using the most relevant form and include:

- the Brain Hands and Node.js versions;
- the operating system;
- a minimal reproduction; and
- redacted error output.

Do not include credentials, private repository content, `.brain-hands/` run
artifacts, or personal data. Security reports belong in a private advisory as
described in [SECURITY.md](SECURITY.md).

Support is community-based and provided without a guaranteed response time.
